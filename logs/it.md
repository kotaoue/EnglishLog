# IT英語

- a firestorme of ~ / 〜 の嵐
- agenda / 議題
- aligned / 足並みをそろえて
- alignment / 連携、協力
- as you see it / 好きなように、ご自由に
- automated / 自動の
- awesome / (口語で)すごい、素晴らしい

- backlog / バックログ。未完成の仕事や対処すべきこと
- be in one's best interest to ~ / 〜するのが最善である
- be inundated with ~ / 〜が殺到する
- benefit / 得をする、利益を得る
- besides / それに、その上
- be up to ~ / 〜に従事して、手掛けて
- bird's-eye view / 「鳥瞰図」の意味から「全体像、概要」
- bleed cash / 現金が流出する。bleedは「出血する、流れる」
- break down ~ into ... /  〜を…に分類する
- burnout / 燃えつきること
