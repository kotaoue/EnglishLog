# workbooks

毎日の英語練習帳を格納するディレクトリです。

ファイル名は `YYYYMMDD.md` の形式（例: `20260222.md`）で、GitHub Actions によって自動生成されます。

## ファイル構成

```sh
workbooks/
├── 20260221.md   # 2026年02月21日分
├── 20260222.md   # 2026年02月22日分
├── ...
└── scoring/
    ├── 20260221_scoring.md   # 2026年02月21日分の採点結果
    ├── 20260222_scoring.md   # 2026年02月22日分の採点結果
    └── ...
```

## 使い方

1. 毎朝 0:00 頃に GitHub Actions が新しい練習帳を PR として作成します
2. PR をマージして `workbooks/YYYYMMDD.md` を開きます
3. 各問題の「あなたの回答:」欄に英語を記入します
4. 翌日、AI が回答を採点してレベルを判定し、次の問題を生成します
